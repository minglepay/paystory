선택한 결제사의 인증화면을 통해 인증을 한 후 결제가 완료되는 일반적인 결제 방식 입니다.

-  ## 빌링 결제
   - ✔ 빌링 방식은 구독 서비스 등의 정기적인 결제가 필요한 경우,<br> PAY-STORY로 부터 구매자의 결제정보가 암호화된 키를 받고 저장하여,<br> 결제가 필요한 시기마다 암호키를 이용하여 결제를 진행할 수 있는 방식입니다.
